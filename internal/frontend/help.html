Homelab AI Help

This chatbot helps you create and manage Minecraft servers on our homelab cluster. Here's everything you can do:

Getting Started

Not sure where to begin? Try one of these:

"What servers exist?"
"What can you help me with?"
"Show me a list of all running servers"

Starting a Server

Ask me to create a vanilla or modded Minecraft server. I'll handle all the setup — Nomad job, DNS, storage, everything.

"Start a new All the Mods 10 server for me"
"Create a vanilla Minecraft server called my-world"
"Provision an ATM10 server with 21000 MHz CPU and 9 GB RAM"

Managing a Server

Restart, stop, start, view logs, or get the connection address for a server.

"Restart my server"
"Stop my-server for tonight"
"Show me the logs for mc-atm10"
"What's the address for my server?"

Player Management

Use RCON commands to manage players on your server.

"Op my friend Alex on my server"
"Deop Steve on mc-atm10"
"Add Alex to the whitelist"
"Who's playing on my server right now?"

Backups

Create and manage world backups to keep your data safe.

"Back up my server"
"Show me the backups for mc-atm10"
"Restore my server from the latest backup"

Teardown

When you're done with a server, I can clean everything up — Nomad job, DNS records, and optionally the files.

"Delete my server and clean up everything"
"Destroy mc-atm10"

Note: I'll always ask for confirmation before deleting anything. Destruction is permanent!

Admin Only

If you're an admin, you have extra powers:

"Show all users' servers"
"Check gateway health status"
"Force-stop mc-atm10"

Keyboard Shortcuts

Enter — Send message
Shift+Enter — New line
Ctrl+/ — Focus input from anywhere
Escape — Cancel in-progress response

Tips
